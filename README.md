## **Tecnologias**

- React Native
- Expo
- Typescript


## **Como executar**
- Clone o repositório
```
git clone https://github.com/JulianaKSCardoso/remindme.git
```

- Instale as dependências

```
npm install
```

- Inicie com
 ```
  expo start
```
